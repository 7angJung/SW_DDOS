# M-GIT

목포해양대학교 컴퓨터 공학과에 재학 중인 학생들을 위한 앱이다. 기본적으로 대학생활에 필요한 기능들인 시간표, 공지사항, 게시판 기능들이 있으며 차별화된 핵심 기능으로는 진로 탐색, 강의 추천, 앞으로 해야 할 활동 등을 알려주는 기능을 가진 앱이다.
    
## Environment
- Java 11
- Gradle
- Android Studio
- Firebase

## Stack
- XML
- Firebase Realtime Database
- Android UI Components
